Simulation-Augmented LLMs for Analog Circuit Design

This project investigates large language models for automated analog circuit design. We develop a closed-loop AI for EDA pipeline where LLMs generate SPICE netlists from natural language specifications, circuits are evaluated through sanity checks, Bayesian optimization-based sizing, and PySpice/SPICE simulations, and the resulting feedback is used to improve model behavior through supervised fine-tuning and reinforcement learning.

  • Closed-loop Circuit Evaluation Pipeline: Built an automated evaluation system for LLM-generated analog circuits, including SPICE netlist generation, sanity checks, topology validation, Bayesian optimization-based sizing, PySpice/SPICE testbench simulation, and extraction of performance metrics such as gain, bandwidth, power, stability, and output behavior.
  • Feedback-driven LLM Training: Transformed simulation errors, sizing results, sanity check outputs, and performance metric gaps into feedback signals for model optimization. The pipeline supports supervised fine-tuning for circuit generation and revision, as well as reinforcement learning using SPICE simulation results as reward signals.
